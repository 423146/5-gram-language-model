# 5-Gram Language Model (n = 5)

Author: Jane Austen  
Dataset: Project Gutenberg

## Description
This project implements a 5-gram language model trained on Jane Austen's works.
The model predicts the next word based on the previous four words.

## How to Run
python ngram_model.py

## Sample Input
The day was very

## Sample Output
the day was very pleasant and she was happy to continue the conversation
